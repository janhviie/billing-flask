Guide To Use.

Links available on Main Page:

1. Add Items: Used to Add new products in the database. You need to enter the item code, item name and other mentioned parameters on the link.
2. New Bill: Used to create a new bill. You need to specify customer information and other details mentioned on the link. After entering you will be directed to bill generation link in which you need to add the items by entering the specific item code and the quantity customer has ordered.
3. Search Bills: Used to search through old bills. You can search bills by entering customer name, item number, mobile number, address, item code or item price.
